<h1 align="middle">찌리리공 뒤집기</h1>

우아한테크코스 8기 프리코스 4주차 오픈 미션을 수행한 저장소입니다.

본 저장소에서는 찌리리공 뒤집기를 구현했습니다.

## 기능 목록

### 게임 관리 기능
- 게임을 생성한다.
  - 게임 코인을 초기화한다.
  - 레벨을 초기화한다.
  - 라운드 번호를 초기화한다.
- 라운드를 생성한다.
  - 보드판을 생성한다.
    - 크기는 5 X 5 격자이다.
    - 각 칸에는 뒤집힌 카드를 놓는다.
    - 카드 구성은 레벨 별 카드 구성 표를 참조한다.
    - 각 행/열에 있는 숫자들의 합과 찌리리공 개수를 각각 계산하여 표시한다.
- 다음의 상황이 발생한 경우, 라운드를 종료한다.
    - 숫자 2, 3 카드가 모두 발견된 경우
    - 찌리리공 카드가 발견된 경우
- 매 라운드가 종료되면 다음의 행동을 수행하고, 다음 라운드를 진행한다.
  - 게임 코인에 획득한 코인을 추가하고, 획득 코인을 1로 초기화한다.
  - 레벨을 업데이트 한다.
    - 찌리리공 카드가 발견된 경우, 1만큼 내린다. 단, 1 미만일 수 없다.
    - 찌리리공 카드가 발견되지 않은 경우, 1만큼 올린다. 단, 6을 초과할 수 없다.
  - 보드판을 다시 생성한다.
- 다음의 상황이 발생한 경우, 게임을 종료한다.
  - 10 라운드가 종료되었을 경우
  - 사용자가 `exit` 명령어를 입력했을 경우

### 카드 뒤집기 기능
- 전달된 위치에 있는 카드를 뒤집는다.
  - 예외 상황:
    - 카드 위치가 A1에서 E5의 범위를 벗어나는 경우
  - 카드 뒤집기에 성공한 경우, 뒤집은 위치를 출력한다.
  ```
  [INFO] A1 카드를 뒤집었습니다.
  ```
  - 이미 뒤집힌 카드의 경우, 안내 메세지를 출력한다.
  ```
  [INFO] 입력한 위치에 있는 카드는 이미 뒤집힌 상태입니다.
  ```

### 카드 마킹 기능
- 전달된 위치에 있는 카드를 마킹한다.
  - 예외 상황:
    - 카드 위치가 A1에서 E5의 범위를 벗어나는 경우
  - 카드 마킹에 성공한 경우, 마킹한 위치와 심볼을 출력한다.
  ```
  [INFO] A1 카드를 숫자 1로 마킹했습니다.
  ```
  - 이미 뒤집힌 카드인 경우, 안내 메세지를 출력한다.
  ```
  [INFO] 입력한 위치에 있는 카드는 이미 뒤집힌 상태입니다.
  ```
### 카드 마킹 해제 기능
- 전달된 위치에 있는 카드가 마킹되어 있는 경우, 마킹을 해제한다.
  - 예외 상황:
    - 카드 위치가 A1에서 E5의 범위를 벗어나는 경우
  - 카드 마킹 해제에 성공한 경우, 마킹 해제한 위치를 출력한다.
  ```
  [INFO] A1 카드의 마킹을 해제했습니다.
  ```
  - 이미 뒤집힌 카드의 경우, 안내 메세지를 출력한다.
  ```
  [INFO] 입력한 위치에 있는 카드는 이미 뒤집힌 상태입니다.
  ```
  - 마킹되어있지 않은 카드인 경우, 안내 메세지를 출력한다.
  ```
  [INFO] 입력한 위치에 있는 카드는 마킹되어있지 않은 상태입니다.
  ```

### 게임 상태 출력 기능
- 누적 코인, 현재 코인, 라운드 레벨 및 보드판 상태를 출력한다. 
```
누적 코인: 2850개 | 현재 코인: 12개 | 레벨: 4

     1    2    3    4    5 
  +----+----+----+----+----+
A |  2 |  1 |  X |  1 |  X |  5 1
B |  3 |  1 |  ? |  ? |  3 |  7 2
C |  ? |  3 |  ? |  2 |  ? |  8 2
D |  1 |  1 |  X |  X |  X |  2 3
E |  X |  1 |  X |  3 |  3 |  7 2
  +----+----+----+----+----+
     5    7    3    6    6 
     3    0    4    2    3
```
### 게임 종료 기능
- 게임 결과를 출력하고 게임을 종료한다.
```
게임이 종료되었습니다.
게임 결과
누적 코인: 3819개
```
